BraillePath

Wearable Navigation Aid for Deafblind Users

A wearable assistive device that combines a refreshable braille display with real-time AI scene detection, helping deafblind users navigate their environment independently. Built with solenoids, a microcontroller, and computer vision — the interface and braille pin mechanism are designed in Autodesk Fusion.

  • Refreshable braille display using solenoids & microcontroller
  • AI-based real-time scene detection
  • Custom braille pin mechanism in Autodesk Fusion
